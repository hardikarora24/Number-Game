GUESS THE NUMBER
This is a simple number guessing game where the player has to guess a number in the given number of tries.The player can select the number of digits in the unknown number and also choose how many tries he wants for guessing.
★ ALL THE DIGITS IN THE UNKNOWN NUMBER ARE UNIQUE(i.e. te number cannot be of the form AABC)
